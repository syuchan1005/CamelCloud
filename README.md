# CamelCloud
## これはなに？
あなたのマシン上で実行できるクラウドストレージ

### インストール

#### コンフィグのコピー

```bash
$ cd ...解凍したzipのPath
$ cp template.config.js config.js
$ $(Your favorite editor) config.js
```

#### クライアントのビルド

```bash
$ cd ...解凍したzipのPath
$ cd Client
$ npm install
$ npm build
```
#### サーバーのビルド

```bash
$ cd ...解凍したzipのPath
$ cd Server
$ npm install
$ npm build
```

### 起動

```bash
$ cd ...解凍したzipのPath
$ cd Server
$ npm start
```
